# CodeAlpha_Credit_Scoring
# Credit Scoring using Machine Learning

## Overview
This project predicts whether a customer is a good or bad credit risk using Machine Learning.

## Objective
To build a model that analyzes customer financial data and predicts creditworthiness.

## Dataset
German Credit Dataset

## Model Used
Random Forest Classifier

## Workflow
- Data preprocessing  
- Label Encoding  
- Train-test split  
- Feature scaling  
- Model training and prediction  

## Tools Used
Python, Pandas, NumPy, Scikit-learn, Jupyter Notebook

## Result
The model successfully predicts customer credit risk with good accuracy.
